 Built With
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- EmailJS (for contact form integration)
- Responsive Design (Flexbox & Media Queries)



Features

- Modern responsive landing page
- Smooth scrolling navigation system
- Mobile-friendly hamburger menu
- Interactive sections (Home, About, Skills, Experience, Projects, Contact)
- Real-world project showcase section
- Contact form with EmailJS integration
- WhatsApp and email contact links
- Downloadable CV button
- Animated UI elements for better user experience

Projects Showcased

- Sky Angels Kiddies Hair Salon Website (skyangels.co.za)
- Labza Co Construction Services Website
- Shepherd Wallpaper Installation Website
- Contact Manager Application
- Jeremiah Chicken & Eggs Business Website
- Mini E-commerce System (cart + WhatsApp checkout)


 Purpose of This Portfolio

This portfolio was built to demonstrate my frontend development skills, showcase real client work, and present my ability to build responsive, functional, and modern websites using HTML, CSS, and JavaScript.

It is also designed to attract freelance clients and job opportunities.



 Responsiveness
